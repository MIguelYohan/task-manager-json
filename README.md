# Gerenciador de Tarefas com POO em Python

Este projeto é um **Gerenciador de Tarefas** simples desenvolvido em Python, focado na aplicação e demonstração de conceitos de Programação Orientada a Objetos (POO) com o intuito de consolidar o aprendizado sobre POO. O módulo permite adicionar, remover, buscar, marcar como concluídas e listar tarefas, persistindo os dados em um arquivo JSON.

Pretendo adicionar um GUI no futuro utilizando pyautogui ou tkinter.
